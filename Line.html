Line

This is a collection and exploration of three interconnected projects. It all began with a chaotic typeface constructed solely from vertical and horizontal lines. From this, a poster and a type specimen were created. Eventually, the project offered a chance to revisit the concept from a new perspective: the typeface poster was recreated digitally, using code to “draw” it entirely with vertical and horizontal lines. At the same time, it was also reinterpreted through an analogue approach, by folding and layering cut paper to build the forms manually.
